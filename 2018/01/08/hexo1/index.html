Hexo + github 搭建个人博客快速上手

最近清闲一些，就把以前想做的事情做了，正好互联网加的服务器到期了，空着一个域名，就把它用来搭建博客了…

#主要步骤
1.前期准备
2.Hexo安装
3.配置Hexo与github page
4.主题更新
5.博客更新

##前期准备
node安装 git安装
*github账户配置

###node安装
node下载
windows: (https://nodejs.org/dist/v9.3.0/node-v9.3.0-x64.msi)
Linux: (https://nodejs.org/dist/v9.3.0/node-v9.3.0-linux-x64.tar.xz)
Mac: (https://nodejs.org/dist/v9.3.0/node-v9.3.0.pkg)
安装没什么大的问题，装好后打开命令行输入
‘node -v’
出现node的版本号说明安装成功了

###git安装
windows: 下载安装(https://git-scm.com/download/win)
Linux和Mac的安装可以参考

###github
既然是利用github page，也得到github官网上注册一个github账号对吧。

##Hexo安装
安装Hexo 初始化Hexo

###安装Hexo
使用NPM进行安装
在命令行中或者任意位置右键,点击git bush here
输入命令
‘npm install hexo -g’

###初始化Hexo
在本地建立你的博客文件夹(如：D：\blog)

进入文件夹，右键运行Git Bush 输入’hexo init’
之后文件夹中会多出这样一些东西

继续在git界面输入
‘hexo g’ >生成Hexo服务器
‘hexo s’ >本地启动（默认4000端口）
在浏览器中输入’localhost:4000’，就可以访问当前的页面了，也可以用来预览Hexo主题
成果出现如下页面，说明Hexo本地部属成功了

##配置Hexo与github page
github账户配置(远程仓库、密钥) Hexo同步

###github账户配置
首先新建仓库，仓库名为你的账户名.github.io
如图

要是github没配置密钥先配置ssh密钥用于链接github远程仓库
任意空白位置右键，git bash here输入命令，引号里输入你的邮箱地址
‘ssh-keygen -t rsa -C “your’s emaill address”‘
回车提交后会提示你文件保存的路径，继续回车确认，提示你输入密码（不可见），再确认输入一次，在刚刚那个路径就会生成id_rsa和id_rsa.pub两个文件
用下面命令复制其内容
‘clip ~<.ssh/id_rsa.pub’
点击个人信息里的设置
(Hexo1/set.png)
点击密钥配置，New SSH key
()
Title随便填，Key里面粘贴刚刚复制的内容，然后点击Add SSH key就OK了
*Hexo同步
到你刚刚建立的博客文件夹，就是启动hexo的地方
记事本打开_config.yml文件，到最后有
‘deploy:
type:’
改成如下图，repository是你的github对应新建的仓库地址，注意在：后面都是有一个空格的，不写会出错

在文件夹空白地方右键git bush命令
‘npm install hexo-deployer-git –save’
安装完成后输入
‘hexo g
hexo d’
然后就可以通过你的github名.github.io访问到你的博客了。
依次输入
‘git config –global user.name “your’s name”
git config –global user.email “your’s email”‘
your’s name 是你的名称，email是你的邮箱
